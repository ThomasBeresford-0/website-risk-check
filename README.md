## UX Flow (Locked)

1. Landing page
   - Single input: website URL
   - One-line explanation of what the scan does
   - Clear disclaimer: not legal advice, no compliance guarantee
   - Primary CTA: “Check my website”

2. Scan preview
   - URL shown
   - “What we’ll check” bullet list
   - Example findings (not results)
   - CTA: “Generate report (£XX)”

3. Payment
   - One-off payment
   - No account creation
   - Clear statement: one-time snapshot, timestamped

4. Report generation
   - Scan runs
   - PDF generated

5. Delivery
   - Instant download
   - Shareable PDF
   - Timestamp visible on report
